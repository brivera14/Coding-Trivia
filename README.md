# CODING - TRIVIA

## PURPOSE
Allow Coding Bootcamp Students to take a timed quiz on JavaScript fundamentals that stores high scores
so that they can gauge their progress compared to their peers.

## BUILT WITH
* HTML
* CSS
* JAVASCRIPT

## Website
https://brivera14.github.io/Coding-Trivia/

## OPERATING INSTRUCTIONS
1. When user click on Start Quiz, Timer start to goes down and a question is presented.
![image](https://user-images.githubusercontent.com/66097407/86089435-39fcd680-ba76-11ea-9994-ba8bcf5365cd.png)
2. If the answer is correct or incorrect It will be presented beneath the multiple choices, and will advance 
to the next questions.
![image](https://user-images.githubusercontent.com/66097407/86089508-5d278600-ba76-11ea-894e-e289c29514c4.png)
![image](https://user-images.githubusercontent.com/66097407/86089548-6c0e3880-ba76-11ea-88c5-48289d8eec51.png)
3. Once all the questions are answerd. It will be display a Game Over screen that will show the score/time left 
and will request the initials to submit.
![image](https://user-images.githubusercontent.com/66097407/86089683-aed01080-ba76-11ea-86db-e7753587684d.png)
4. If user click on submit button without enter initials. It will be display an alert.
![image](https://user-images.githubusercontent.com/66097407/86089747-d3c48380-ba76-11ea-8bb2-f4422fb350e8.png)
5. Once the user enter intitial and click on submit button. It will be display the High Score screen 
with the initials and score that are in localStorage.
![image](https://user-images.githubusercontent.com/66097407/86089849-08d0d600-ba77-11ea-9254-6cd525a3dcbb.png)
6. To go back to the Start Quiz screen just need to click on "Go Back" Button.

## Note:
If the timer reach zero "0" and user has not answerd all the questions, the quiz will display automatically the last Initials and score stored.
